# 影音下載器 Pro (MP3 Downloader)

一個強大、簡潔且具備**自動更新核心**功能的影音下載工具。
基於 `yt-dlp` 與 `FFmpeg` 強大核心開發，不僅支援 YouTube，還能處理 Facebook、Instagram、Twitch 等主流平台，甚至能突破部分自架站或加密串流的限制。

## ✨ 特色功能

*   **多格式支援**：
    *   🎵 **MP3**：一般音樂下載 (自動轉檔壓縮)。
    *   🎼 **WAV**：無損音訊格式。
    *   🎬 **MP4**：最高畫質影片與音訊自動合併。
*   **自動核心更新** 🔄：
    *   程式啟動時自動檢查並下載最新的 `yt-dlp` 核心。
    *   **再也不怕 YouTube 改版導致下載器失效！**
*   **強大相容性** 💪：
    *   內建 `FFmpeg`，無需使用者額外安裝設定。
    *   **無視 SSL 錯誤**：支援下載憑證過期或自簽的私有伺服器影片。
    *   **偽裝瀏覽器**：自動帶入 User-Agent 與 Referer，突破防盜連限制。
*   **直覺介面**：圖形化操作 (GUI)，支援進度條顯示與自訂下載路徑。

## 🚀 如何使用

1.  **執行程式**：直接打開 `Mp3_D.exe` (首次執行需等待幾秒下載核心)。
2.  **貼上網址**：將影片或音樂連結貼入輸入框。
3.  **設定選項**：
    *   (選填) 自訂檔案名稱。
    *   選擇格式 (MP3 / WAV / MP4)。
    *   選擇下載資料夾。
4.  **開始下載**：點擊按鈕，等待進度條跑完即可！

## 🛠️ 進階技巧：下載「彈出式」或「無法解析」的影片

如果直接貼上網址無法下載（例如某些教學網站、彈出式視窗或自架影音站），請嘗試以下 **「F12 大法」**：

1.  在瀏覽器開啟該影片頁面。
2.  按下鍵盤 **`F12`** 開啟開發者工具。
3.  切換到 **「Network (網路)」** 分頁。
4.  在過濾框 (Filter) 輸入 `m3u8` 或 `mp4`。
5.  重新整理網頁並播放影片。
6.  若列表中出現 `.m3u8` 結尾的連結：
    *   對該連結點擊 **右鍵** -> **Copy** -> **Copy link address** (複製連結網址)。
7.  將複製的 `m3u8` 網址貼回本下載器，通常即可成功下載。

> 若仍無法成功，代表該網站使用了更高階的 DRM 加密 (如 Widevine)，這類影片目前無法透過下載器取得。

## 📦 技術棧 (Tech Stack)

*   **語言**：Python 3.11
*   **GUI 框架**：Tkinter
*   **核心引擎**：[yt-dlp](https://github.com/yt-dlp/yt-dlp) (透過 subprocess 呼叫)
*   **媒體處理**：[FFmpeg](https://ffmpeg.org/)
*   **打包工具**：PyInstaller

## 💻 如何自行開發與打包 (Development & Build)

如果你想修改程式碼或自行打包 EXE，請參考以下步驟：

1.  **安裝依賴套件**：
    ```bash
    pip install -r requirements.txt
    ```

2.  **準備 FFmpeg**：
    *   請至 [FFmpeg 官網](https://ffmpeg.org/download.html) 下載 Windows 版本。
    *   解壓縮後，將 `bin` 資料夾內的 `ffmpeg.exe` 與 `ffprobe.exe` 複製到專案根目錄。

3.  **執行打包指令**：
    ```bash
    pyinstaller Mp3_D.spec
    ```
    打包完成後，執行檔將位於 `dist/Mp3_D.exe`。
